# Erik Lujan | Backend Developer Portfolio

Este repositorio contiene el código fuente de mi portfolio profesional, una aplicación web diseñada para presentar mi trayectoria, stack tecnológico y los proyectos más relevantes desarrollados en el ámbito del backend y la ingeniería de datos.

La plataforma está construida bajo una arquitectura moderna utilizando **Angular** para la interfaz de usuario y **Tailwind CSS** para un diseño responsivo y optimizado.

## Sobre mí

Soy **Técnico Universitario en Programación** graduado de la Universidad Tecnológica Nacional (UTN). Mi enfoque técnico se centra en el ecosistema de Python, especializándome en la construcción de APIs RESTful escalables, el diseño de bases de datos relacionales y la automatización de procesos mediante pipelines de datos.

## Proyectos Destacados

El portfolio presenta una selección de soluciones técnicas que demuestran mi capacidad para resolver problemas complejos de negocio:

* **[StockFlow](https://github.com/ErikLujan/stockflow-sistema-gestion-de-inventario):** Sistema integral para la gestión de inventario. Implementa una arquitectura full-stack con una API de alto rendimiento en FastAPI y un panel de administración dinámico en Angular.
* **[Task Queue System](https://github.com/ErikLujan/Task-Queue-System):** Infraestructura para el procesamiento de tareas asíncronas en segundo plano. Utiliza Redis como message broker para optimizar los tiempos de respuesta y garantizar la escalabilidad del sistema.
* **[Tech Job Reporter](https://github.com/ErikLujan/tech-job-reporter):** Pipeline ETL automatizado diseñado para el análisis del mercado laboral IT. Realiza la extracción, procesamiento con Pandas y generación de reportes analíticos distribuidos mediante plantillas dinámicas.
* **[Job Scraper API](https://github.com/ErikLujan/job-scraper-project):** Motor analítico contenedorizado en Docker para la extracción y normalización de ofertas laborales. Utiliza operaciones de *upsert* en Supabase para asegurar la integridad de la persistencia de datos.

## Stack Tecnológico

### Backend & Core
* **Lenguajes:** Python (Asyncio), TypeScript, SQL.
* **Frameworks:** FastAPI, Flask.
* **Gestión de Dependencias:** uv, pip.

### Frontend (Portfolio)
* **Framework:** Angular.
* **Estilos:** Tailwind CSS.

### Infraestructura & Datos
* **Bases de Datos:** PostgreSQL (Supabase), Redis, SQLite.
* **Herramientas:** Docker, Git, GitHub Actions.
* **Deployment:** Render, Vercel.
